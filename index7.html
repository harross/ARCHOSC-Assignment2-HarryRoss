OS Scheduling

First, I implemented long-term scheduling by representing each process as a Job object and admitting only those with a priority greater than 5 using a LINQ filter:

var jobs = new List<Job> {
    new Job { Name = "A", Priority = 5,  ArrivalTime = 0,  ExecutionTime = 3 },
    new Job { Name = "B", Priority = 4,  ArrivalTime = 2,  ExecutionTime = 6 },
    new Job { Name = "C", Priority = 8,  ArrivalTime = 5,  ExecutionTime = 5 },
    new Job { Name = "D", Priority = 6,  ArrivalTime = 6,  ExecutionTime = 3 },
    new Job { Name = "E", Priority = 10, ArrivalTime = 8,  ExecutionTime = 6 },
    new Job { Name = "F", Priority = 3,  ArrivalTime = 9,  ExecutionTime = 2 },
    new Job { Name = "G", Priority = 7,  ArrivalTime = 10, ExecutionTime = 6 }
};

var admittedJobs = jobs.Where(job => job.Priority > 5).ToList();

Jobs C, D, E, and G are admitted; A (priority 5), B (4), and F (3) are rejected:
